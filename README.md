# EMG-and-HR-data-workflow
Данный репозиторий содержит python notebook с примерами обработки данных ЭМГ и ЧСС из экспериментов по гедонистическому восприятию животных и растительных белков.

Эксперимент заключался в регистрации физиологических показателей (электромиографии мимическим мышц и частоты сердечных соркащений) во время дегутсации участниками проб мяса животного и растительного происхождения. Целью эксперимента было выявление физиолоических паттернов, связанных с восприятием белков животного и растительноого происхождения. Глобально данная работа была посвящена разработке подхода к объективной оценке вкусового восприятия для будующего тестирования продуктов питания и предсказания их успешности на рынке.
